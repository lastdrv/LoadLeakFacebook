Простой пример загрузки утекших телефонов русского сегмента facebook в БД.

Для сборки запускаем build.sh
Создаём таблицу из create_table.sql
В окружении нужно иметь переменные HOST, USER и PASS, ну или написать руками в application.properties
Переходим в каталог app и запускаем start.sh

Шаблон данных в facebook.txt
В оригинале ~10 млн строк

P.S. Реальных телефонов тут конечно не будет
